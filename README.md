#  Pokétrivial

App interactiva creada para una sección de un podcast.

Ruleta animada con sistema de luces, marcador por equipos y secuencia 
de 6 minijuegos garantizada (cada juego aparece 2 veces antes del juego preparado para el final (combate)).

**Demo en vivo:** https://rodrigo-corvi.github.io/Trivial-poke/

Construido con Vanilla JS y Canvas API.
